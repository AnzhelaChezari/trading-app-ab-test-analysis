# Анализ влияния онбординга на финансовое поведение пользователей (A/B тест) 📈

Проект посвящен исследованию гипотезы в сфере FinTech: поможет ли обучение финансовой грамотности в процессе онбординга удержать пользователей и увеличить количество повторных депозитов.

## 🛠 Технологический стек
*   **Python:** Pandas, NumPy, Matplotlib, Seaborn.
*   **Статистический анализ:** 
    *   **Bootstrap** (анализ распределений, работа с перцентилями).
    *   Проверка статистической значимости изменений (p-value).
*   **Продуктовая аналитика:** Воронки (Funnel analysis), сегментация, анализ депозитарного поведения.


## 📊 Структура данных
*   `Исследование результатов АВ-теста и поиск инсайтов.ipynb`: основной ноутбук с расчетами и графиками.
*   `conclusions`: Основные результаты, которые помогут улучшить продукт.
